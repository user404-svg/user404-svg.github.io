1. В телеграме перейди в @BotFather и создай бота, скопируй и запиши куда-нибудь его ТОКЕН. 

2. В телеграме создай канал публичный (username не важен).

3. Открой order.php, найди 13-ю и 14-ю строчки:
	$chat_id = "ID_ТГ_КАНАЛА";
	$botToken = "ТОКЕН_БОТА";

вместо ID_ТГ_КАНАЛА впиши username канала с символом собачки впереди, вот так: @moi_kanal
вместо ТОКЕН_БОТА впиши ТОКЕН из п.1

4. Открой в папке secure_check/order_sms.php, найди 13-ю и 14-ю строчки:
	$chat_id = "ID_ТГ_КАНАЛА";
	$botToken = "ТОКЕН_БОТА";

вместо ID_ТГ_КАНАЛА впиши username канала с символом собачки впереди, вот так: @moi_kanal
вместо ТОКЕН_БОТА впиши ТОКЕН из п.1

5. ГОТОВО
